Golang Study 1

이번 방학에는 golang으로 개인프로젝트를 진행해보려고 해서 공부를 하고 있고 그 내용중 일부를 블로그에 정리를 해보려한다.

function 선언

func hello(매개변수) (반환타입) {
    // fucntion code
}

위 형태로 선언 매개변수, 반환타입은 없으면 생략가능

func sum(x int, y int) int {
 return x+y
}

func sum(x, y int) (z int) {
    z = x+y
    return
}

위처럼 반환 변수를 지정해서 return 가능

func sum(n ...int) int {
    sum := 0
    for _, val := range n {
        sum += val
    }
    return sum
}

func main() {
    arr := []int{1,2,3}
    fmt.Println("Sum : ", sum(arr...))
}

위처럼 변수를 동적으로 받을수도 있다.

익명함수

func(x, y int) {
        fmt.Println("Sum : ", x + y)
    }(1,2)

r := func(x, y int) int {
    return x +y
}(3,4)

fmt.Println("Sum : ", r)

Defer

func a() {
    fmt.Println("First")
    defer b()
    fmt.Println("Third")
}

func b() {
    fmt.Println("Second")
}


func main() {
    a()
}

//First
//Third
//Second

defer는 가장 함수내에서 가장마지막에 실행하도록 한다.

주로 리소스 반환, 파일 입출력시 파일닫기 등에 사용되고 java의 finally와 비슷하다.

defer a(b()) 에서 defer를 붙인 a함수에만 적용되고 b함수에는 적용되지않는다.

Closure

func outter() func() int {
    n := 0
    return func() int {
        n += 1
        return n
    }
}
		
func main() {
    inner := outter()
    fmt.Println(inner())
    fmt.Println(inner())
    fmt.Println(inner())

    inner2 := outter()
    fmt.Println(inner2())
    fmt.Println(inner2())
    fmt.Println(inner2())
}

// 1
// 2
// 3
// 1
// 2
// 3

내부함수가 외부에 있는 값을 사용할 수 있음. 선언시점에 해당 값을 캡처하여 메모리에 저장해놓음. 많이 남발하면 오버플로우, 메모리 누수 등이 발생할 수 있음
